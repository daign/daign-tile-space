# daign-tile-space

[![NPM package][npm]][npm-url]

Data structure for a recursive tile raster.

## Installation

```sh
npm install @daign/tile-space --save
```

## Scripts

#### Build

    npm run build

#### Run lint analysis

    npm run lint

#### Run unit tests with code coverage

    npm run test
